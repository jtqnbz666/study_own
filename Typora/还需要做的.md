面试题
[30, 50, 50]
112元， 可重复选， 用递归的方式做(把动态规划的公司代入即可)

dp[n] = max(dp[n], dp[n - coins[i]] + coins[i]);

递归的函数格式为
void dfs(money, coins) {
	for (int i = 0; i < coins.size(); i ++) {
		if (money > coins[i]) {
			result = max(result, dfs(money - coins[i]) + coins[i]);
		}
	}
	
	return result
}
